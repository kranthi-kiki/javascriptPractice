using jQuery

jQuery is a JavaScript Library that focuses on simplifying DOM manipulation, AJAX calls, and Event handling. It is used by JavaScript developers frequently.
jQuery uses a format, $(selector).action() to assign an element(s) to an event.
To explain it in detail, $(selector) will call jQuery to select selector element(s), and assign it to an event API called .action()

$ - to difine or access jQuery;
(selector)- to find html element
$(" selector").action() : action - to be performed
